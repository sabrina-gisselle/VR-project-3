Project 3 - James Terrell Art Exhibition

The user can walk around a museum that is modeled after the Guggeinhiem. The users can go into four rooms that replicate the work of James Terrell and his work with light. Unlike the typical rules of the museum this environment allows you to alter the art either by color change or change in position.

Insight goals: Users will appreciate light in a new way which is the main goal of James Terrell. Users will experience James Terrell's art work and appreciate how light can be deceiving.

Components To Help With Simulation Sickness

  • Didn't use HUD
  • Removed head bobbing functionality
  • Added different texture to the floor vs walls so the user knows their orientation
  • Added signs for each room to again orient the user and give them more information.
  • Minimal Lag -- Because we are dealing with light we knew that flicker would be harmful so the scene has fewer assets but greater detail on the light components.
